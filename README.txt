SCRATCH QUEST — SIMPLE TIMELINE SITE (v2)

Files in this folder:
- index.html  (page layout + bio + logo)
- styles.css  (styles)
- data.js     (your timeline entries — edit this to update)
- logo.jpg    (your branding)

How to update the live site:
1) Copy these files into your repo folder named YOURUSERNAME.github.io (replace the old ones).
2) In GitHub Desktop: Commit to main -> Push origin.
3) Refresh YOURUSERNAME.github.io after ~1–2 minutes.

To add entries later: open data.js, add another object in the ENTRIES list, Save, Commit & Push.
